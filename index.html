League of Legends Build Tester

How to Use

Before using this tool, play 10 matches with the same build.

That can be as simple as playing the same champion, or more complicated such as using the same runes with that champion. You can even use the same items if desired.

Due to the fact that both certain runes and items may be changed per game depending on the conditions, how strict you want the testing to be is left up to you; only the winrate is returned.
