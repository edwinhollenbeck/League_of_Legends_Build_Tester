League of Legends Build Tester

About the App

This app is intended for current ranked players of League of Legends, and is designed to assist with gauging the effectiveness of champion builds. Riot's matchmaking system is designed to keep all players more or less at a 50% win-rate, so any build providing a higher win-rate than that is generally considered to be good. Up to ten ranked matches per champion that you want to test need to be played before using the tool. So decide what build you'd like to test and then play as many matches, up to ten, that you want to use for testing. The more matches you play the more accurate the results will be. A build includes the runes selected, as well as items used if you want to get more specific.

How to Use

Enter your Summoner name and select a champion you want to test for from the drop-down list. Once you select the Go! button the tool will calculate your win-rate from the last 10 matches played with that champion, less if you haven't played that many matches yet. Once you get your result you can select the Test Again button to test another champion's build or retest the same champion after playing more matches with the same build or a new one.
